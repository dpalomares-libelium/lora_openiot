# LoRa OpenIOT code

This repo contains sample code for the LoRa network setup:

  - Code for the Arduino sensor nodes
  - Code for the LoRa gateway (sensor concentrators)
  - Libraries needed

For setting up the LoRa module libraries read this [tutorial].
